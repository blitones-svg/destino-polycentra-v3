# Destino de Polycentra V3

Mini game em HTML, CSS e JavaScript pronto para compartilhamento.

## Como abrir

Abra o arquivo `index.html` em um navegador moderno.

## O que precisa permanecer junto

- `index.html`
- `styles.css`
- `script.js`
- pasta `assets/`

## Publicar online

Você pode publicar esta pasta diretamente em:

- GitHub Pages
- Netlify
- Vercel

## Observações

- A música pode exigir um clique do usuário para começar, por política do navegador.
- O progresso salvo usa `localStorage`, então fica armazenado no navegador de cada pessoa.
